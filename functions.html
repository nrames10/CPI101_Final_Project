A List of Website Functions to Implement

Tools: Html,css, javascript

Homepage: A home page contains proportal to each function, by default it will show most popular recipes. When access to the link it will first show a window of welcome information and ask a user to answer some questions like if they prefer to share or read recipes and if they are new to cooking. Using this information, the website will automatically link them to the area which is most suitable for them. They can also choose to not answer the question and then go to the

Sharing Recipes: Users can share recipes by words, pictures and videos. Any posting should include at least one picture. Users can comment, like, repost or save a recipe. Views: People can choose to view only certain categories, for example, recipes for newbies or from friends.

Account: Functions to log in with email and set up personal information. People need to login in order to share recipes. Each person will have a personal page showing their personal information that they choose to share and a page of the recipes they posted. People can add friends and view only the recipe from friends.
